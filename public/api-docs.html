VIDO 开放接口文档

面向合作方 / 内部系统的对接文档。通过 AppID + AppKey 签名访问平台全部核心能力：网剧生成、图生视频、数字人、剧情创作、声音克隆。

Base URL：/openapi · 协议：HTTPS · 编码：UTF-8 · 签名：HMAC-SHA256

概览

所有对外开放接口的 URL 以 /openapi 为前缀，对应内部 /api 的能力。每次请求必须携带签名 header：X-App-Id、X-Timestamp、X-Nonce、X-Signature。

账号由平台管理员在管理后台「接口账号」中创建，创建时勾选可调用的接口范围。每个账号独立的 AppID/AppKey + 独立的接口权限 + 调用配额。

签名认证

请求 Header

Header说明示例
X-App-Id管理后台分配的 AppIDak_0c8d…
X-TimestampUNIX 秒级时间戳（±300 秒内有效）1776180000
X-Nonce客户端生成的随机字符串（16-32 字节，五分钟内不重复）a1b2c3d4e5…
X-SignatureHMAC-SHA256 签名结果（见下）f27b3a…
Content-TypeJSON 请求需为 application/jsonapplication/json

签名算法

拼接待签名字符串 payload：

payload = METHOD + "\n" + PATH + "\n" + TIMESTAMP + "\n" + NONCE + "\n" + BODY_HASH
  • METHOD — 大写 HTTP 方法：GET / POST / PUT / DELETE
  • PATH — 原始请求路径（含 /openapi，不含 query string），例 /openapi/drama/projects
  • TIMESTAMP — 与 header X-Timestamp 一致
  • NONCE — 与 header X-Nonce 一致
  • BODY_HASH — SHA-256(raw_body).hex()；GET / 无 body 请求传空字符串 ""

签名：

signature = HMAC_SHA256( app_secret, payload ).hex()
⚠️ AppKey 仅在创建/重置时显示一次，请立即妥善保存。AppKey 泄露会导致账号被盗用，可在管理后台「重置 AppKey」。

示例代码

Node.js

const crypto = require('crypto');
const axios = require('axios');

const APP_ID  = 'ak_xxx';
const APP_KEY = 'xxxxxxxx';
const BASE    = 'https://your-domain.com';

async function callApi(method, path, body) {
  const ts = Math.floor(Date.now() / 1000).toString();
  const nonce = crypto.randomBytes(16).toString('hex');
  const rawBody = body ? Buffer.from(JSON.stringify(body)) : Buffer.alloc(0);
  const bodyHash = rawBody.length ? crypto.createHash('sha256').update(rawBody).digest('hex') : '';
  const payload = [method.toUpperCase(), path, ts, nonce, bodyHash].join('\n');
  const signature = crypto.createHmac('sha256', APP_KEY).update(payload).digest('hex');

  return axios({
    method, url: BASE + path, data: body || undefined,
    headers: {
      'X-App-Id': APP_ID, 'X-Timestamp': ts, 'X-Nonce': nonce,
      'X-Signature': signature, 'Content-Type': 'application/json',
    }
  }).then(r => r.data);
}

// 示例：创建网剧
callApi('POST', '/openapi/drama/projects', {
  title: '朔风惊魂', synopsis: '古代将军穿越现代...', episode_count: 10
}).then(console.log);

Python

import hmac, hashlib, time, secrets, json, requests

APP_ID  = 'ak_xxx'
APP_KEY = 'xxxxxxxx'
BASE    = 'https://your-domain.com'

def call_api(method, path, body=None):
    ts = str(int(time.time()))
    nonce = secrets.token_hex(16)
    raw = json.dumps(body, separators=(',', ':'), ensure_ascii=False).encode('utf-8') if body else b''
    body_hash = hashlib.sha256(raw).hexdigest() if raw else ''
    payload = '\n'.join([method.upper(), path, ts, nonce, body_hash])
    signature = hmac.new(APP_KEY.encode(), payload.encode(), hashlib.sha256).hexdigest()
    headers = {
        'X-App-Id': APP_ID, 'X-Timestamp': ts, 'X-Nonce': nonce,
        'X-Signature': signature, 'Content-Type': 'application/json'
    }
    return requests.request(method, BASE + path, data=raw if raw else None, headers=headers).json()

print(call_api('POST', '/openapi/drama/projects', {
    'title': '朔风惊魂', 'synopsis': '古代将军穿越现代...', 'episode_count': 10
}))

cURL

# 准备变量
APP_ID="ak_xxx"
APP_KEY="xxxxxxxx"
PATH_="/openapi/drama/suggest-scene-params"
METHOD="POST"
TS=$(date +%s)
NONCE=$(openssl rand -hex 16)
BODY='{"synopsis":"古代将军穿越现代","style":"水彩手绘"}'
BODY_HASH=$(printf '%s' "$BODY" | openssl dgst -sha256 -hex | awk '{print $2}')
PAYLOAD="${METHOD}
${PATH_}
${TS}
${NONCE}
${BODY_HASH}"
SIG=$(printf '%s' "$PAYLOAD" | openssl dgst -sha256 -hmac "$APP_KEY" -hex | awk '{print $2}')

curl -X $METHOD "https://your-domain.com${PATH_}" \
  -H "X-App-Id: $APP_ID" \
  -H "X-Timestamp: $TS" \
  -H "X-Nonce: $NONCE" \
  -H "X-Signature: $SIG" \
  -H "Content-Type: application/json" \
  -d "$BODY"

错误码

HTTP含义排查
401签名错误 / AppID 不存在 / 时间戳过期检查 payload 拼接、时间戳同步、nonce 是否重复
403账号已停用 / 接口无权限管理后台确认账号状态和接口勾选
404未知接口路径参考下方接口清单
429速率限制（上游）退避重试
500服务内部错误查看响应 body 的 error 字段
统一返回格式：{ success: boolean, data?: any, error?: string }
