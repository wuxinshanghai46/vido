# VIDO AI 数字人与视频生成操作指南

## 一、完整操作指南：如何生成数字人和视频

### 方式 A：生成 AI 视频（剧情动画）

全自动流水线，从文字到成片：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 打开 http://localhost:3007 | Studio 主页 |
| 2 | 左面板「剧本」tab → 输入主题/描述 | 如"古风仙侠少女修仙" |
| 3 | 选择风格：2D / 3D / 写实 | 决定画面风格 |
| 4 | 点「生成」 | 自动走完 5 步流水线 |

**流水线 5 步：**

```
剧本生成 → 角色/场景图生成 → 视频片段生成 → 语音合成 → FFmpeg 合成终片
```

---

### 方式 B：生成数字人视频（口播/广告）

| 步骤 | 操作 | API |
|------|------|-----|
| 1 | 上传人像照片（正面清晰半身照） | `POST /api/avatar/upload-image` |
| 2 | 输入口播文案（或用 AI 生成） | `POST /api/avatar/generate-text` |
| 3 | 选择背景（办公室/演播室/户外等） | 预设 5 种 + 自定义 |
| 4 | 选择视频模型 + TTS 语音 | 见下方模型表 |
| 5 | 点生成 → SSE 实时进度 | `POST /api/avatar/generate` |

**数字人生成原理：**

```
人像图片 → i2v模型(让人物动起来) → TTS(文字转语音) → FFmpeg(视频+音频合成) → 成品
```

## 二、工作流系统 + 需要调用的模型

### 工作流节点（可视化拖拽）

打开 http://localhost:3007/workflow.html，支持以下节点类型：

```
[文本节点] → [角色节点] → [视频节点] → [语音节点] → [合成节点]
                ↗                          ↗
[背景节点]                    [音乐节点]
```

| 节点 | 功能 | 调用的模型 |
|------|------|-----------|
| 文本节点 | 剧本/文案生成，支持短剧、动作、广告、品牌文案模板 | DeepSeek / OpenAI / Claude |
| 角色节点 | 生成角色形象图 | 即梦AI / 智谱 / OpenAI / Stability |
| 背景节点 | 生成场景背景图 | 同上 |
| 视频节点 | 文生视频 / 图生视频 | 智谱 / MiniMax / Kling / Sora 等 |
| 语音节点 | TTS 文字转语音 | 讯飞 / 火山 / Fish Audio / OpenAI |
| 音乐节点 | BGM 背景音乐混合 | 本地上传 |
| 合成节点 | 多片段拼接 + 转场 | FFmpeg（本地） |

### 工作流 API

| 端点 | 说明 |
|------|------|
| `POST /api/workflow/save` | 保存/创建工作流 |
| `GET /api/workflow/:id` | 加载工作流 |
| `GET /api/workflow` | 工作流列表 |
| `DELETE /api/workflow/:id` | 删除工作流 |
| `POST /api/workflow/execute` | 自动执行流水线 |
| `GET /api/workflow/execute/:taskId` | 查询执行进度 |
| `POST /api/workflow/effects/apply` | 应用后期特效 |
| `POST /api/workflow/refine-video-prompt` | AI 优化视频提示词 |

## 四、生成抖音广告数字人（实战教程）

以生成类似"淘宝闪购/外卖优惠"广告口播为例：

### 第 1 步：准备人像

**方式一：** 上传自己的照片（正面半身照、光线均匀、背景简洁）

```bash
curl -X POST http://localhost:3007/api/avatar/upload-image \
  -F "image=@my_photo.jpg"
# 返回: { "filename": "xxx.jpg", "path": "/api/avatar/images/xxx.jpg" }
```

**方式二：** 使用系统预设形象

```bash
curl http://localhost:3007/api/avatar/presets
# 返回预设形象列表：商务女性、新闻主播等
```

### 第 2 步：AI 生成广告文案

```bash
curl -X POST http://localhost:3007/api/avatar/generate-text \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "淘宝闪购今日活动，外卖只需几块钱起",
    "template": "promo",
    "duration": 30
  }'
# 返回 300-500 字的广告口播稿
```

支持的文案模板：

| 模板 | 说明 |
|------|------|
| `promo` | 商品促销（300-500字） |
| `knowledge` | 知识科普 |
| `news` | 新闻播报 |
| `story` | 故事叙述 |
| `tutorial` | 教程讲解 |

### 第 3 步：自动分段（多表情多动作）

```bash
curl -X POST http://localhost:3007/api/avatar/segment-script \
  -H "Content-Type: application/json" \
  -d '{
    "text": "家人们！今天给大家带来一个超级福利！淘宝闪购今日活动...",
    "segmentDuration": 10
  }'
# 返回自动分段结果，每段 8-12 秒，含推荐表情和动作
```

### 第 4 步：生成数字人视频

**单段式（简单）：**

```bash
curl -X POST http://localhost:3007/api/avatar/generate \
  -H "Content-Type: application/json" \
  -d '{
    "imageUrl": "/api/avatar/images/xxx.jpg",
    "text": "家人们！今天给大家带来一个超级福利！淘宝闪购今日活动，外卖只需几块钱起！",
    "background": "studio",
    "model": "cogvideox-flash",
    "voiceGender": "female"
  }'
```

**多段式（推荐，不同表情动作更自然）：**

```bash
curl -X POST http://localhost:3007/api/avatar/generate \
  -H "Content-Type: application/json" \
  -d '{
    "imageUrl": "/api/avatar/images/xxx.jpg",
    "background": "studio",
    "model": "MiniMax-Hailuo-2.3",
    "voiceGender": "female",
    "segments": [
      {
        "text": "家人们！今天给大家带来一个超级福利！",
        "expression": "excited",
        "motion": "挥手打招呼，面带兴奋表情"
      },
      {
        "text": "淘宝闪购今日活动，外卖只需几块钱起！",
        "expression": "smile",
        "motion": "双手比出OK手势，微笑点头"
      },
      {
        "text": "点击下方链接，直接下单，手慢无哦！",
        "expression": "excited",
        "motion": "指向下方，竖起大拇指"
      }
    ]
  }'
```

**表情选项：** `natural` | `smile` | `serious` | `excited` | `calm`

### 第 5 步：查看进度和结果

```bash
# SSE 实时进度
curl http://localhost:3007/api/avatar/tasks/{taskId}/progress

# 播放视频
http://localhost:3007/api/avatar/tasks/{taskId}/stream

# 下载视频
http://localhost:3007/api/avatar/tasks/{taskId}/download
```

### 第 6 步：后期特效（可选）

通过工作流添加文字、产品图、BGM：

```bash
curl -X POST http://localhost:3007/api/workflow/effects/apply \
  -H "Content-Type: application/json" \
  -d '{
    "videoPath": "outputs/avatar/xxx.mp4",
    "effects": {
      "textOverlays": [{ "text": "限时特惠 ¥9.9", "position": "bottom" }],
      "bgm": { "url": "/path/to/bgm.mp3", "volume": 0.3 }
    }
  }'
```

## 五、最低配置 & 推荐配置

### 最低成本（可免费起步）

| 功能 | 平台 | 费用 |
|------|------|------|
| 剧本/文案 | DeepSeek | 极低（百万 token 几元） |
| 视频/数字人 | 智谱 CogVideoX-Flash | 有免费额度 |
| 语音 | Windows SAPI | 免费（质量一般） |

### 推荐质量方案

| 功能 | 平台 | 原因 |
|------|------|------|
| 剧本/文案 | DeepSeek 或 Claude | 中文质量好 |
| 数字人视频 | MiniMax Hailuo 2.3 或 Kling v3 | 画质最好 |
| 语音 | 火山引擎 或 Fish Audio | 抖音同款 / 支持克隆 |
| 角色图 | 即梦 AI 或 智谱 | 中文优化 |

### 配置方法

1. 打开 http://localhost:3007 → 设置页面
2. 添加供应商 → 选择平台预设 → 填入 API Key
3. 添加模型 → 设置用途（story / image / video / tts）
4. 测试连接确认可用
